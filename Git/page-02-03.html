Prérequis

Pour lire ce livre, aucune connaissance des systèmes de gestion de versions ou de développement n’est nécessaire, bien qu’une expérience de ces systèmes puisse faciliter la compréhension de certains concepts.

Tous les exemples de ce livre utilisent l’interface en ligne de commande, le lecteur devra donc être habitué à celle-ci pour suivre ces exemples.
