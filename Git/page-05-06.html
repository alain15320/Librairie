Outils pour travailler avec Markdown

Du fait de sa grande popularité, Markdown s’est vu doté de nombreux outils au fil des années. Les développeurs, les blogueurs et même les auteurs de l’édition traditionnelle se sont intéressés au format, d’où la grande diversité d’outils. 

1. Sublime Text

Ce n’est pas l’outil qui gère le mieux le Markdown, en revanche, c’est l’éditeur que de nombreux développeurs utilisent quotidiennement. Il est donc normal de vouloir l’utiliser pour écrire la documentation également. Sublime Text ne gère pas nativement le Markdown (hormis le fait que ce soit un éditeur de texte), en revanche il existe plusieurs plug-ins permettant d’éditer facilement et visuellement le Markdown.

Sublime Text est un éditeur de texte compatible avec les trois systèmes d’exploitation principaux, Linux, Mac OS et Windows.

Le plug-in Markdown Preview permet d’exporter facilement le contenu d’un fichier Markdown vers un fichier HTML. C’est un système très pratique pour versionner une documentation et partager une version plus lisible. Voir la page GitHub du produit : https://github.com/revolunet/sublimetext-markdown-preview

Le plug-in MarkdownEditing permet d’avoir une interface proposant la coloration syntaxique. Cette interface se montre plus intuitive et permet de faciliter la lecture de documents Markdown bruts.

Le plug-in MarkdownTOC (pour Markdown Table Of Content) permet d’éditer une table des matières en fonction des titres présents dans le document. L’un des avantages de ce plug-in est qu’après export en HTML (avec Markdown Preview), il est possible de cliquer sur un élément de la table des matières pour accéder à la section pointée. Il est d’ailleurs possible de préciser une profondeur de titre à intégrer. Cette fonctionnalité permet par exemple de n’intégrer dans la table des matières que les titres de niveau 1 et 2.

2. Texts

Texts est un logiciel de bureautique qui permet de créer des documents qui seront stockés au format Markdown. Il est utilisable sur les systèmes Mac OS et Windows.

L’une des particularités de cet éditeur vient de sa compatibilité d’importation et d’exportation. Texts est capable d’importer des fichiers Microsoft Office Word, HTML, Latex ou encore OPML.

Ce logiciel est capable d’exporter les documents Markdown en documents Microsoft Office Word, PDF, HTML, ePub. Être capable d’exporter les documents Markdown dans un grand nombre de formats permet d’utiliser Markdown en interne et d’utiliser un autre format pour la communication externe tout en gardant la flexibilité offerte par Markdown.

Pour les exports, ce logiciel nécessite l’installation de Pandoc, un convertisseur de documents texte.

Texts est vendu 30 dollars sur le site officiel : http://www.texts.io.

3. Ulysses

Ulysses est une application pour Mac OS et iOS. Elle présente plusieurs avantages très intéressants :

  • Des formats variés d’export (HTML, ePub, PDF, docx).

  • Une gestion avancée de bibliothèque personnelle. Il est possible d’organiser ses écrits facilement.

  • Plusieurs styles d’export sont proposés et il est possible d’utiliser des styles personnalisés.

  • D’autres fonctionnalités sont particulièrement intéressantes pour les auteurs et les blogueurs mais elles dépassent le cadre de cet ouvrage.

Ulysses sauvegarde également automatiquement l’espace de travail régulièrement. Il est possible de récupérer un ancien fichier comme une documentation qui semblait inutile à une époque et qui a été supprimée.

Il autorise la personnalisation de la syntaxe du Markdown interprété. Cela permet notamment de changer la manière de spécifier un bloc de code ou d’autres syntaxes très couramment utilisées. Pour cela, il faut cliquer sur le menu Ulysses puis Préférences et pour finir sur Balisage. Ci-dessous se trouve une capture d’écran de la fenêtre qui permet cette personnalisation.

Pour personnaliser la syntaxe, il faut tout d’abord créer un nouveau balisage puis double cliquer sur les éléments pour définir la syntaxe et un raccourci si nécessaire.

La réelle valeur ajoutée pour un développeur est la qualité des exports. Un développeur novice ne trouvera pas l’export austère et pourra lire la documentation de manière fluide.

Ulysses est vendu 44,99 euros sur l’Apple Store.
