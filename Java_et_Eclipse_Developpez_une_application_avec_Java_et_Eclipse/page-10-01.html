Introduction

Ce chapitre nécessite des connaissances générales sur les bases de données et sur le langage SQL (Structured Query Language).

L’application XAMPP, acronyme de X Apache MySQL Perl PHP, est utilisée. Un serveur, une base de données relationnelle et une interface graphique d’administration sont ainsi immédiatement disponibles. Les langages Perl et PHP ne seront pas utiles dans le cadre du projet Luna. Il existe des versions de XAMPP pour Windows, Linux, Mac OS et Solaris.

Vous avez aussi la possibilité de choisir Wamp.
