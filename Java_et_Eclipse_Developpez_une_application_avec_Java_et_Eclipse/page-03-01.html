Installation du JDK Java 8

 Vérifiez auparavant la présence et la version de Java.

Sous Windows, selon les versions, vous pouvez procéder de différentes manières. Le plus simple est d’utiliser un des sites dépendant d’Oracle, par exemple : http://java.com/fr/download/installed.jsp

Si vous ne disposez pas encore de la version 8 pour les développeurs, vous pouvez la télécharger sur le site d’Oracle. L’accès aux ressources Java n’étant actuellement pas pratique sur ce site, voici l’URL d’accès direct à ce jour : http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

 Acceptez l’agrément de licence puis choisissez la version du JDK qui correspond à votre système d’exploitation : Windows, Linux, Solaris ou Mac OS.

Pour écrire le projet sous Windows 8 64 bits, la version Windows x64 a été choisie.

 Procédez à l’installation du JDK en double cliquant sur l’exécutable.

Vous pouvez utiliser les options par défaut de l’installateur, elles sont bien adaptées pour l’utilisation d’Eclipse.

Il est indispensable de disposer de l’aide. Celle-ci est accessible en ligne sur le site d’Oracle : http://download.oracle.com/javase/8/docs/api/

Vous pouvez aussi la télécharger actuellement à l’adresse : http://www.oracle.com/technetwork/java/javase/documentation/jdk8-doc-downloads-2133158.html

 Procédez à l’installation puis double cliquez sur le fichier index.html dans le dossier tutorial pour consulter l’aide en mode déconnecté (offline).
