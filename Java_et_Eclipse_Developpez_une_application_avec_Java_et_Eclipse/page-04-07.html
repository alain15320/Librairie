Premier exécutable

Les sources d’une application sont essentielles aux développeurs. Pour les utilisateurs, c’est le binaire (l’exécutable) sur lequel ils double cliquent pour lancer l’application qui est important.

La création de l’exécutable de la classe PremiereFenetre se fait grâce aux étapes suivantes.

 Choisissez l’option Export dans le menu File d’Eclipse. Une boîte de dialogue s’ouvre.

 Cliquez sur le bouton Next.

 Sélectionnez dans la liste déroulante la configuration de lancement de votre application, ici PremiereFenetre - MonPremierProjet.

 Choisissez ensuite le nom du fichier qui sera généré, la seule contrainte étant que le nom du fichier finisse par l’extension .jar, en cliquant sur le bouton Browse....

 Cliquez sur le bouton Finish.

Il peut arriver qu’une boîte de dialogue d’avertissement s’affiche. Ce n’est peut-être pas forcément très grave, mais prenez l’habitude de lire les messages en cliquant sur Details.

 Vérifiez enfin votre application en double cliquant sur le fichier .jar qui vient d’être généré.
