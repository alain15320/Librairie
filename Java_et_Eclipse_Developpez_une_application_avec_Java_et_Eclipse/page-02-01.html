Introduction

Java 8 est disponible depuis le 18 mars 2014. Le célèbre langage de programmation orienté objet de SUN, racheté par Oracle en avril 2009, serait utilisé à ce jour par quelque neuf millions de développeurs dans le monde et présent sur plus de 95 % des ordinateurs.

Pour les développeurs, plusieurs versions sont disponibles sur le site d’Oracle (ou ceux dépendant d’Oracle : openjdk.java.net, java.net) pour couvrir les principales plateformes : Linux, Solaris et Windows. Apple propose pour l’heure sa propre implémentation du JDK. Cet ouvrage utilise le JDK 8 version 64 bits pour Windows.

Le logiciel utilisé pour développer avec Java 8 est Eclipse IDE for Java Developers. Cette version inclut WindowBuilder qui permet de créer simplement et rapidement de belles IHM (interfaces homme-machine). Pour créer des états (appelés également rapports) personnalisés et des graphiques, l’outil de reporting JasperReports a été choisi. Bien que le livre ait été écrit avec la version 4.4 d’Eclipse (dite Luna), son contenu ainsi que le projet développé sont compatibles avec la version 4.5 d’Eclipse (dite Mars).
